# Movie-Recommender-System
A Web Based user-item Movie Recommendation Engine using Content Based Recommender System By vectorization and
The recommendation based on the underlying idea that is movie will be recommended based on the tags provided by user.

##### Requirements
```
python 3.6

pip3

```
##### Setup to run

```
To install Dependencies

```
pip install -r requirements.txt
```

### Creating Local Server

Goto directory, example

```
cd ../Movie-Recommendation
```
To run
```
streamlit run app.py  or py -m streamlit run app.py
```
Now open your browser and go to this address
```
http://localhost:8501/
```
